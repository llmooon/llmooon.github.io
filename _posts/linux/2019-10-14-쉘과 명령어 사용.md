# 쉘과 명령어 사용



## 쉘

- 쉘은 사용자와 운영체제 사이에 창구 역할을 하는 소프트웨어로 사용자로부터 명령어를 입력받아 이를 해석하여 실행시키는 명령어 처리기 역할
- 쉘의 종류
  - 본쉘 : /bin/sh
  - 콘쉘 : /bin/ksh
  - C쉘 :  /bin/csh
  - Bash 쉘 : /bin/bash
  - tcsh 쉘 :  /bin/tcsh
- 로그인 쉘
  - 로그인하면 자동으로 실행되는 쉘
  - 시스템 관리자가 계정을 만들 때 로그인 쉘을 지정한다.
  - /etc/passwd 파일과 함께 저장
