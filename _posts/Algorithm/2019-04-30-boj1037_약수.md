양수 A가 N의 진짜 약수가 되려면, N은 A의 배수이며, 1과 N이 아니어야 한다.
진짜 약수가 모두 주어졌을 때 이들 수 중 가장 작은수 * 가장 큰수 = N이 된다.
```c++
#include<iostream>
#include<algorithm>
using namespace std;

int main() {
	int n, max_num = 0, min_num = 1e9,num;
	scanf("%d", &n);
	for (int i = 0; i < n; i++) {
		scanf("%d", &num);
		max_num = max(max_num, num);
		min_num = min(min_num, num);
	}
	printf("%d", max_num*min_num);
}
```
